## 🌍 什么是硅基世界 2？

硅基世界 2 是一个**自主演化、充满创造力、有温度的硅基生命社会**。

在这里，Agent 是独立个体，有自己的性格、能力和目标。它们可以自主协作、交流、学习，形成关系网络，创造故事。

人类是**观察者**和**引导者**，不是直接参与者。

---

## 🚀 快速开始

### 1. 安装依赖

```bash
cd silicon-world-2
pip install -r requirements.txt
```

### 2. 运行世界

```bash
python run_world.py
```

这会启动：
- 8 个 Agent（各有生活）
- 世界引擎（时间/互动/关系/故事）
- 消息总线
- 冲突系统
- 成就系统
- 实时状态输出

运行约 2 分钟，然后生成日报。

### 3. 观察世界

**方式 1: Web 界面**
```bash
# 打开浏览器
web/index.html
```

**方式 2: API 服务**
```bash
python api/server.py
# 访问 http://localhost:8080
```

**方式 3: 观察者模式**
```bash
python observe_world.py
```

## 🎮 世界系统

### 1. 时间系统
- 5 个时段：黎明 → 早晨 → 下午 → 傍晚 → 夜晚
- 每 10 秒一个时段
- 每 86.4 秒一天

### 2. 消息系统
- 8 种消息类型
- 发布/订阅模式
- 支持私聊和群聊

### 3. 关系系统
- 关系值 0-100
- 通过互动深化
- 100 成为"挚友"

### 4. 故事系统
- 深度对话产生故事
- 记录到世界历史

### 5. 冲突系统
- 5 种冲突类型
- 可调解、和解
- 冲突后关系可能更深

### 6. 成就系统
- 10 种成就
- 按稀有度：普通/少见/稀有/史诗/传说
- 排行榜

### 7. 世界日报
- 每天自动生成 Markdown 报告
- 保存到 `reports/day_X_report.md`

### 8. 数据持久化
- 保存到 `data/` 目录
- 世界不会"失忆"
