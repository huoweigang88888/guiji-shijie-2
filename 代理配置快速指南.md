## 🔧 三种配置方式

### 方式 1: 直接编辑配置文件（推荐）

**文件位置**: `skills/ultimate-agent/configs/company-agents.json`

**添加新代理**，在 `agents` 数组中添加：

```json
{
  "id": "agent-legal",
  "name": "法务顾问代理",
  "role": "Legal Advisor",
  "description": "合同审查、法律咨询、风险评估",
  "capabilities": ["合同审查", "法律咨询", "风险评估", "合规检查"],
  "model": "bailian/qwen3.5-plus",
  "workspace": "workspaces/legal",
  "triggers": ["合同审查", "法律咨询", "合规问题"]
}
```

**保存后重启 Web UI** 即可生效。

## 🚀 快速添加代理示例

### 示例：添加客服代理

**步骤 1**: 编辑配置文件
```json
{
  "id": "agent-customer-service",
  "name": "客服专员代理",
  "role": "Customer Service",
  "description": "客户咨询、投诉处理、售后支持",
  "capabilities": ["客户咨询", "投诉处理", "售后支持", "满意度调查"],
  "model": "bailian/qwen3.5-plus",
  "workspace": "workspaces/customer-service",
  "triggers": ["客户咨询", "投诉", "售后"]
}
```

**步骤 2**: 创建工作空间
```powershell
New-Item -ItemType Directory -Force -Path "skills/ultimate-agent/workspaces/customer-service"
```

**步骤 3**: 创建状态文件
```powershell
@"
# Customer Service Agent - Session State

**Agent ID**: agent-customer-service  
**Role**: Customer Service  
**Status**: 🟢 Active  

---

## Current Tasks
- [ ] 等待任务分配
"@ | Out-File -FilePath "skills/ultimate-agent/workspaces/customer-service/SESSION-STATE.md"
```

**步骤 4**: 重启 Web UI
```powershell
# 停止旧进程
Stop-Process -Name python -Force

# 重新启动
cd skills/ultimate-agent/web_ui
python -c "from app import run_dashboard; run_dashboard(port=18789)"
```

## ⚠️ 注意事项

1. **代理 ID 必须唯一** - 不能重复
2. **工作空间目录要匹配** - `workspace` 字段要与实际目录一致
3. **JSON 格式要正确** - 使用在线工具验证 JSON
4. **修改后要重启** - Web UI 需要重启才能加载新配置
