## 🎬 备用方案

### 如果演示出现问题

1. **数据未生成**
   ```bash
   # 检查运行状态
   ls -l demo_data.json
   
   # 重新运行
   python run_demo_30min.py
   ```

2. **Web 界面无法访问**
   ```bash
   # 检查端口
   netstat -ano | findstr :8000
   
   # 更新 Web 服务
   cd web
   python server.py
   ```

3. **数据异常**
   ```bash
   # 查看原始数据
   python -c "import json; print(json.load(open('demo_data.json')))"
   ```
