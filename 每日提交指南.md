# 🤖 硅基世界 2 - 每日自动提交

## ✅ 快速使用

### 每天手动提交一次（最简单）

在项目目录中运行：

```bash
auto-commit.bat
```

或者用 PowerShell：

```powershell
.\auto-commit-daily.ps1 -RunNow
```

## 📅 自动定时提交配置

### Windows 任务计划程序（推荐）

1. 按 `Win + R`，输入 `taskschd.msc`，回车
2. 右侧点击 **"创建基本任务"**
3. 名称：`SiliconWorld2-DailyCommit`
4. 触发器：选择 **"每天"**
5. 时间：`02:00`（凌晨 2 点）
6. 操作：选择 **"启动程序"**
7. 程序：`C:\Windows\System32\cmd.exe`
8. 参数：
   ```
   /c "cd /d C:\Users\97RPZ8T\.openclaw\workspace\silicon-world-2 && auto-commit.bat"
   ```
9. 完成 ✓

## 📊 查看提交状态

```bash
# 查看最近的提交
git log --oneline -5

# 查看文件状态
git status

# 查看远程仓库
git remote -v
```

## 🔧 其他命令

```powershell
# 立即提交一次
.\auto-commit-daily.ps1 -RunNow

# 查看定时任务状态
.\auto-commit-daily.ps1 -Status

# 删除定时任务
.\auto-commit-daily.ps1 -Remove
```

## 📝 提交消息格式

自动提交的消息格式：
```
🤖 自动提交 - 2026-04-07 10:25
```

或手动提交时：
```bash
git commit -m "✨ 添加了新功能"
```

## ⚠️ 注意事项

1. **确保网络畅通** - 需要访问 GitHub
2. **首次需要登录** - 确保 Git 已配置 GitHub 凭证
3. **检查提交结果** - 定期查看 GitHub 仓库确认提交成功
4. **敏感信息** - 不要提交密码、密钥等敏感数据

## 🛠️ 故障排除

### Git Push 失败
```bash
# 测试 GitHub 连接
git push -v origin main

# 检查 Git 配置
git config --list
```

### 查看定时任务日志
1. 打开 **事件查看器** (eventvwr.msc)
2. 导航到：**应用程序和服务日志** → **Microsoft** → **Windows** → **TaskScheduler** → **Operational**
