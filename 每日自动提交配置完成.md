## 🚀 使用方法

### 方法一：每天手动运行（推荐）

**最简单方式**：双击运行
```
silicon-world-2\每日提交.bat
```

或在项目目录运行：
```bash
auto-commit.bat
```

### 方法二：设置 Windows 定时任务

1. 按 `Win + R`，输入 `taskschd.msc`
2. 创建基本任务
3. 名称：`SiliconWorld2-DailyCommit`
4. 每天 凌晨 2:00
5. 启动程序：`C:\Windows\System32\cmd.exe`
6. 参数：
   ```
   /c "cd /d C:\Users\97RPZ8T\.openclaw\workspace\silicon-world-2 && auto-commit.bat"
   ```

### 方法三：使用 PowerShell 命令

```powershell
# 立即提交一次
.\auto-commit-daily.ps1 -RunNow

# 设置定时任务（需要管理员权限）
.\auto-commit-daily.ps1 -Setup

# 查看状态
.\auto-commit-daily.ps1 -Status
```

## 📊 提交流程

```
检查 Git 状态
    ↓
发现更改？
    ├─ 否 → 显示"无需提交"
    └─ 是 → 显示更改列表
            ↓
        添加所有更改 (git add -A)
            ↓
        提交更改 (git commit)
            ↓
        推送到 GitHub (git push)
            ↓
        完成 ✓
```

## ⚠️ 注意事项

1. **GitHub 凭证** - 确保已配置 Git 和 GitHub 认证
2. **网络连接** - 提交时需要访问 GitHub
3. **敏感信息** - 不要提交密码、密钥等
4. **定期检查** - 确认定时任务正常运行
