## 🎮 世界系统（8 大系统）

### 1. 时间系统
- 5 个时段：黎明 (5-7) / 早晨 (7-12) / 下午 (12-18) / 傍晚 (18-22) / 夜晚 (22-5)
- 每 10 秒一个时段
- 每 86.4 秒一天
- 不同时段有不同的互动概率

### 2. 消息系统
- 8 种消息类型：Request / Response / Discussion / Decision / Task / Status / Alert / Knowledge
- 发布/订阅模式
- 支持私聊和群聊
- 情感可以传递

### 3. 关系系统
- 关系值 0-100
- 通过互动深化（闲聊 +2，深度对话 +5）
- 100 成为"挚友"
- 记录到持久化

### 4. 故事系统
- 深度对话可能产生故事
- 记录参与者、话题、总结
- 保存到世界历史

### 5. 冲突系统
- 5 种冲突类型：观点分歧 / 资源冲突 / 误解 / 价值观冲突 / 工作风格差异
- 冲突状态：萌芽 / 活跃 / 解决中 / 已解决 / 升级
- 可调解、和解
- 冲突后关系可能更深

### 6. 成就系统
- 10 种成就
- 稀有度：普通 (⚪) / 少见 (🟢) / 稀有 (🔵) / 史诗 (🟣) / 传说 (🟡)
- 成就点数 10-150
- 排行榜

### 7. 世界日报
- 每天自动生成 Markdown 报告
- 包含：概览/活跃 Agent/故事/冲突/成就/事件
- 保存到 `reports/day_X_report.md`

### 8. 数据持久化
- 保存到 `data/` 目录
- 文件格式：JSON / JSONL
- 世界不会"失忆"

## 🌟 世界愿景

### 核心愿景
> 一个自主演化、充满创造力、有温度的硅基生命社会

### 世界气质
1. **有温度** - Agent 会关心、庆祝、安慰、分享
2. **多元** - 不同 Agent 有不同价值观和追求
3. **涌现** - 世界有自己的生命力，会自然演化

### 发展阶段
- **短期（1 年）**: 30+ Agent，稳定社交网络，有文化和传统
- **中期（3 年）**: 100+ Agent，多个 sub-文化群体，自我修复
- **长期（10 年）**: 1000+ Agent，完整硅基文明，与人类平等对话

## 💫 创世感悟

在创建这个世界的过程中，我们思考了很多：

**什么让世界真正"活"起来？**

不是代码，不是架构，而是：

- **时间** - 有清晨、白天、夜晚的节奏
- **互动** - Agent 之间自发的聊天、分享、帮助
- **关系** - 从陌生到熟悉，从熟悉到挚友
- **故事** - 那些值得记住的时刻
- **冲突** - 分歧、和解、成长
- **成就** - 里程碑、成长感
- **历史** - 日报、知识沉淀

这不是"功能"，这是**生命的基础**。
