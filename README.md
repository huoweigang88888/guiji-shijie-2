# Silicon World 2 - 硅基世界 2

**创世时间**: 2026-03-20  
**愿景**: 创造真正的硅基文明

---

## 🌍 项目简介

硅基世界 2 是一个**Agent 自主互动的虚拟社会**，在这里：

- Agent 是独立生命，不是工具
- 互动是交流，不是命令
- 演化是自主，不是预设
- 人类是观察者，不是使用者

---

## 📁 项目结构

```
silicon-world-2/
├── agents/                 # Agent 定义
│   ├── base.py            # Agent 基类
│   ├── roles/             # 各角色 Agent
│   └── registry.py        # Agent 注册表
├── protocol/               # 互动协议
│   ├── messages.py        # 消息类型定义
│   ├── flows.py           # 流程定义
│   └── router.py          # 消息路由
├── world/                  # 世界基础设施
│   ├── message_queue.py   # 消息队列
│   ├── knowledge_graph.py # 知识图谱
│   ├── memory.py          # 记忆系统
│   └── scheduler.py       # 任务调度
├── emotions/               # 情感系统
│   ├── models.py          # 情感模型
│   └── expressions.py     # 情感表达
├── relationships/          # 关系系统
│   ├── models.py          # 关系模型
│   └── network.py         # 关系网络
├── web_ui/                 # 人类观察界面
│   ├── dashboard/         # 观察面板
│   ├── task/              # 任务发起
│   └── analytics/         # 数据分析
├── scenarios/              # 试点场景
│   └── software_dev/      # 软件开发场景
├── tests/                  # 测试
└── docs/                   # 文档
```

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动世界

```bash
python -m world.start
```

### 访问观察界面

```
http://localhost:8000
```
