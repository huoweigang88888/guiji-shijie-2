- 代理状态监控

## 🚀 技术选型

### 方案 A: 轻量级（推荐）
- **Web 框架**: Flask
- **模板引擎**: Jinja2
- **前端**: 原生 HTML + CSS + JS
- **图表**: Chart.js
- **优点**: 简单、快速、无需构建
- **缺点**: 功能相对简单

### 方案 B: 现代化
- **Web 框架**: FastAPI
- **前端**: Vue.js 或 React
- **UI 组件**: Element Plus 或 Ant Design
- **构建工具**: Vite
- **优点**: 现代化、功能强大
- **缺点**: 复杂度高、需要构建

### 方案 C: 嵌入式
- **Web 框架**: Eel 或 PyWebView
- **前端**: 任意 Web 技术
- **优点**: 像桌面应用
- **缺点**: 打包复杂

**推荐**: 方案 A（快速开发，满足需求）
