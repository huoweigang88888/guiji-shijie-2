### 任务 3.3: 用户确认点 ✅

**目标**: 实现阶段完成后的用户确认机制

**执行结果**:
- ✅ `require_confirm` 字段支持
- ✅ `on_confirm` 回调函数
- ✅ 确认前暂停执行
- ✅ 用户可拒绝进入下一阶段

**确认流程**:
```
阶段完成
    ↓
检查 require_confirm
    ↓
是 → 调用 on_confirm() → 用户确认 → 继续
                      ↓
                    用户拒绝 → 暂停工作流
    ↓
否 → 直接进入下一阶段
```

**验收**: ✅ 通过
