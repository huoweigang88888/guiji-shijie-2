## ⚠️ 已知限制

### 当前版本 (v0.1.0-alpha)

1. **Flask 依赖** - 需要手动安装 `pip install flask`
2. **无 WebSocket** - 使用轮询而非实时推送
3. **只读模式** - 大部分功能仅展示，不能操作
4. **无认证** - 无登录/权限控制
5. **无移动端优化** - 响应式但未针对触摸优化

### 待开发功能
