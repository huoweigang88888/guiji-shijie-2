# 项目

**创建时间**: 2026-03-17

## 快速开始

### 环境要求
- Python 3.11+
- PostgreSQL 15+

### 安装
```bash
pip install -r requirements.txt
```

### 运行
```bash
python app/main.py
```

### 测试
```bash
pytest tests/
```

## 项目结构
```
src/
├── app/
│   ├── main.py
│   ├── models/
│   ├── routes/
│   └── services/
└── tests/
```
