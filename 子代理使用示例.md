# 子代理使用示例

**说明**: 演示如何使用已配置的 5 个子代理

---

## 📋 可用的子代理

| 代理 | 职责 | 使用场景 |
|------|------|----------|
| analyst | 数据分析师 | 数据分析、报表生成 |
| research-assistant | 研究助理 | 信息搜索、调研 |
| agent-coordinator | 代理协调员 | 多代理协作 |
| executor | 任务执行器 | 一次性任务 |
| message-aggregator | 消息聚合器 | 消息管理 |

---

## 🚀 使用方法

### 方法 1: Python 代码

```python
from openclaw import sessions_spawn

# 使用分析师
result = sessions_spawn(
    task="分析 2026 年 Q1 销售数据",
    label="analyst",
    runtime="subagent",
    mode="session"
)
```

### 方法 2: 命令行

```bash
openclaw sessions spawn --task "分析数据" --label analyst
```

### 方法 3: 在聊天中指定

```
@analyst 请分析这份销售报告
@research-assistant 搜索最新的 AI 技术
```

## 📊 实际示例

### 示例 1: 使用分析师

**任务**: 分析销售数据

**输入**:
```
@analyst 请分析以下销售数据：
- Q1 销售额：100 万
- Q2 销售额：150 万
- Q3 销售额：120 万
- Q4 销售额：180 万

请计算增长率并给出建议。
```

**预期输出**:
```markdown
# 销售分析报告

## 数据概览
- 全年总销售额：550 万
- 平均季度销售额：137.5 万

## 增长率分析
- Q1→Q2: +50%
- Q2→Q3: -20%
- Q3→Q4: +50%
- 全年增长率：80%

## 建议
1. Q3 出现下滑，需要分析原因
2. Q4 表现强劲，可以延续策略
3. 建议制定 Q1 促销计划
```

## 💡 最佳实践

### 1. 明确任务描述
```
❌ 模糊："分析数据"
✅ 明确："分析 2026 年 Q1-Q4 销售数据，计算增长率，给出 3 条建议"
```

### 2. 提供必要信息
```
❌ 不完整："调研报告"
✅ 完整："调研 2026 年 AI 技术，包括框架、突破、应用，输出 Markdown 报告"
```

### 3. 指定输出格式
```
"请用 Markdown 格式输出报告"
"请列出 3-5 条建议"
"请提供数据表格"
```
