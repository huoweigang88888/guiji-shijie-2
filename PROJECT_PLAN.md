## 🛠️ 技术栈

- **主语言**: Python 3.14+
- **记忆系统**: elite-longterm-memory + fluid-memory
- **搜索 API**: OpenClaw web_search / Brave Search
- **消息平台**: WhatsApp / Telegram / Slack
- **版本控制**: Git + GitHub
- **部署**: OpenClaw 技能系统

---

## 📝 开发规范

1. **代码风格**: PEP 8 + 类型注解
2. **提交信息**: 清晰描述变更内容
3. **文档**: 每个模块必须有 docstring
4. **测试**: 核心功能需要单元测试
5. **日志**: 使用结构化日志记录关键操作

---

## 🔐 安全注意事项

- 不存储敏感凭证在代码中
- 使用环境变量或加密存储
- 外部 API 调用需要错误处理
- 定期审查和更新依赖
